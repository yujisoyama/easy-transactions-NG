NG Desafio

Aplicação web fullstack, dockerizada, cujo objetivo seja possibilitar que usuários da NG consigam realizar transferências internas entre si.

▶️ Rodando localmente

Pré-requisitos:

  • Ter o Docker instalado na sua máquina, pois toda a aplicação será subida em containers no Docker.

Passo a Passo:

  • Clone esse repositório:
git clone https://github.com/yujisoyama/ngdesafio.git
  • Entre na pasta do projeto:
cd ngdesafio
  • Vamos construir os containers da aplicação server-side e da aplicação client-side. Para isso, execute os dois comandos abaixo:
docker compose build server

docker compose build web
  • Finalizando a construção dos dois containers, execute o seguinte comando para rodar toda a aplicação:
docker compose up
  • Para acessar a interface web após a subida dos containers, entre no endereço abaixo:
http://localhost:5137/)

🛠 Stacks utilizadas

Front-end:

  • React: utilizei o React, seus Hooks, Context API e algumas de suas bibliotecas para fazer toda a interface e manipulação de dados em seus componentes.
  • TypeScript: utilizei para tipar a aplicação através de interfaces.
  • TailwindCSS: utilizei o TailWind CSS na estilização. Acho o Tailwind com o React uma boa combinação, pois a componentização do React impede em estilizações muito longas junto com o código.
  • Axios: utilizei para o consumo de API’s.

Back-end:

  • Node e Express: desenvolvimento server-side e a criação de suas rotas.
  • TypeScript: utilizei para tipar a aplicação através de interfaces.
  • TypeORM: integração da aplicação com o PostgreSQL através de classes entidades, consultas, e design da arquitetura do banco de dados.
  • JWT: JsonWebToken para criação de seção do usuário.
  • Bcrypt: hashear a senha salva no banco.
